Discord NBA Bet Bot

Discord Bot where users can bet on NBA games with fake server currency using Las Vegas odds. Used SQLite3 database in order to keep track of players and bets. I wrote this program because my friends and I wanted an easy way to mock-bet on NBA games and keep track of our predictions.

DupeFinder

Application that allows users to find duplicate images at a certain similarity level. Finds similarity between images by running an average hashing algorithm. I wrote this program to help some of my photographer friends find image duplicates that they had saved that were filtered or resized.

Technologies used: Python, PyQt5
